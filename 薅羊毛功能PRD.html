一、需求背景

D助 App 内有大量分散在不同页面的每日福利任务（每日签到、网吧打卡、口令码兑换等），用户容易遗漏。本次新增「薅羊毛」功能，将福利任务聚合到一个页面，提供进度追踪和完成奖励。

3.8 订阅号消息推送 [新增]

推送触发机制

核心逻辑：后台更新任务配置时触发推送，每日最多推送 1 次。
不是定时推送，而是运营在业务字典中更新 woolTaskConfig 时，由后端判断今日是否已推送过，未推送则推送一次。
例如：运营在周一、周三、周五更新任务，则用户一周只会收到 3 条推送。
项目说明
推送渠道通过「活动小助手（待定）」订阅号向已订阅用户推送羊毛提醒
触发时机后台更新 woolTaskConfig 配置时触发，每日最多推送 1 次
去重规则后端记录每日推送标记（key: wool_push_{yyyy-MM-dd}），更新时检查：
• 今日已推送 → 跳过，不重复推送
• 今日未推送 → 执行推送，写入标记
推送对象在薅羊毛页面开启铃铛订阅的用户
推送内容取任务列表中排序第一条（sort 最大）的任务信息作为推送内容
推送样式卡片式消息，包含封面图、标题、摘要、跳转入口
跳转逻辑用户点击「前往薅羊毛」跳转至薅羊毛页面

推送触发流程

运营更新 woolTaskConfig（业务字典）
  → 后端接收配置变更回调
  → 检查功能开关 open 是否为 1
  → 检查今日是否已推送（查 wool_push_{今日日期}）
    → 已推送：跳过
    → 未推送：
      → 取 tasks 中 sort 最大的第一条任务
      → 组装推送内容（标题/正文/封面图）
      → 向已订阅用户批量推送
      → 写入今日推送标记

推送消息卡片结构

📸 {封面图}
{标题大字}
活动小助手（待定）
{推送标题}
{推送正文}
📋 今日可薅：{数量}个 🎁 奖励：{奖励名称}
前往薅羊毛 > 查看详情

模板字段说明

字段数据来源说明必填
{封面图}取 tasks 中第一条（sort 最大）有 banner 的图片；无则用后端配置的默认封面图推送卡片封面图是
{标题大字}后端推送模板配置封面图上大标题，如「今日羊毛已就位！」是
{推送标题}{标题大字} - {第一条任务名}如「今日羊毛已就位！- 助手每日签到」是
{推送正文}后端推送模板，自动替换第一条任务的 name 和 reward如「快来薅羊毛～首个任务：助手每日签到，可得深渊票 x100」是
{任务数量}自动统计当日 status=1 且在有效期内的任务数如「3个」是
{奖励名称}读取 bonusReward.name100% 达标奖励是
前往薅羊毛固定文案跳转至薅羊毛页面固定
封面图逻辑：遍历 tasks（按 sort 降序），取第一个 banner 不为空的图片。全部为空则用后端默认封面图。
标题/正文逻辑：取排序第一的任务 name + reward 与推送模板拼接。
频次保障：即使运营一天内多次更新配置，用户也只会收到 1 条推送。

4.3 tasks[] 字段说明

字段类型必填说明
idNumber是唯一标识
nameString是任务名称
tagsArray否标签，含 text + color（red/orange/purple）
rewardString是奖励描述
linkString是跳转链接
bannerString否图标URL，空则显示🎁
sortNumber否排序权重，越大越前
refreshFreqString是daily=每日 / weekly=每周 / once=仅一次 / custom=自定义
refreshDaysNumber否仅 custom 生效。其他频次时此字段不生效
statusNumber是1=上线，0=下线
startTimeString否生效时间，空=立即
endTimeString否失效时间，空=长期

4.4 bonusReward 字段说明

字段类型必填说明
nameString是奖励名称
contentString否奖励详情（弹窗展示）
iconString否奖励图标URL
giftCodeString是礼包单号
requiredDaysNumber是本周需薅羊毛天数，达标后可领取奖励。如 5 表示本周完成 5 天即可领奖
giftLimitObject是礼包限量：hour/day/week/month/total，0=不限
userLimitObject是单用户限量：hour/day/week/month/total，0=不限
startTimeString否奖励生效时间，格式 YYYY-MM-DD HH:mm:ss，为空立即生效
endTimeString否奖励失效时间，为空长期有效。过期后前端不展示领取按钮
statusNumber是1=启用，0=停用
