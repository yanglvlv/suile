# 2026-03-25 工作日志

## 薪酬数据看板 - 字段来源逻辑分析页面
- 基于 `看板指标 (1).xlsx` 梳理了51个字段的数据来源逻辑
- 创建了完整的HTML数据看板页面：`.codebuddy/数据看板/index.html`
- 字段来源分为6类：薪酬导入、活动创建、系统接入、公式计算、匹配关联、无
- 关键字段来源：
  - **薪酬导入类**：订单创建月份(薪酬月份)、平台、账号ID(平台账号ID)、已执行总价含税(薪酬金额)
  - **活动创建类**：订单号(上游订单号)、开始/结束时间
  - **匹配关联类**：账号名称(通过ID匹配昵称)、有效阅读量/播放量/互动量等(通过字段名近似匹配，如"访问人数≈UV")
  - **公式计算类**：CPE=薪酬/(赞+评+转)、CPM=薪酬/播放×1000、CPC=薪酬/播放、观看成本/点击成本/互动成本
- 页面功能：筛选、排序、分页、导出Excel（含来源说明行）、字段来源可视化面板
