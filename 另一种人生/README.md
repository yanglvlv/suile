# 🌆「另一种人生」—— 沙盒人生模拟器

## 一、项目概述

一款 H5 网页端的开放式人生模拟游戏，参考《模拟人生》和《GTA》的自由度设计。
玩家在一座虚拟城市中自由生活，所有结果由多个系统联动运算产生，不写死数据。

**核心理念**：不写死任何东西，所有结果都是系统运算出来的。

## 二、七大核心系统

### 1. 🧬 角色属性引擎
- **身体层**：体力、健康度、饥饿度、疲劳度、外貌值
- **精神层**：快乐度、压力值、孤独感、灵感值、意志力、情绪状态
- **社会层**：声望、信用、魅力
- **经济层**：现金、存款、负债、资产、月收支
- 属性公式驱动，互相影响（如连续加班→疲劳飙升→免疫力降→概率生病）

### 2. 🕐 时间流系统
- 一天 6:00~次日2:00，连续时间轴
- 每个行为消耗真实时长
- 白天/黑夜、工作日/周末、四季轮换、节假日
- 天气系统（晴/阴/雨/暴风雨/雪/雾/大风），影响心情和活动

### 3. 🗺️ 城市生态系统（14个地点）
- 🏠 我的家、🏢 写字楼、☕ 街角咖啡、🏋️ 健身房
- 🌳 城市公园、📚 图书馆、🍜 美食街、🛒 购物中心
- 🍺 酒吧、💼 人才市场、🏫 培训学校、🏥 医院
- 🏦 银行、🏪 便利店
- 每个地点有开放时间、可用行动、可能出现的NPC
- 地点分区域（老城区/CBD/市中心/文创区），通勤有时间消耗

### 4. 👥 NPC 关系网络（8个初始NPC）
- 每个NPC有独立性格模型（外向/善良/幽默/野心）
- 四维好感度：信任/亲密/尊重/喜欢
- NPC有自己的日程表，不同时间出现在不同地点
- 关系等级：陌生人→认识→朋友→密友
- NPC有记忆系统，记得你的互动历史
- 社交互动：打招呼、闲聊、深聊、请客、送礼、求助、一起玩

### 5. 💰 经济/投资系统
- **股票市场**：10支股票，按行业分类，K线波动由算法驱动
  - 受市场情绪、经济周期、游戏内事件影响
  - 有内幕消息机制（关系好的NPC透露行业消息）
- **房产投资**：不同区域房价有涨跌，可自住/出租
- **基金理财**：稳健/平衡/激进三种风格
- **彩票**：每周开奖，小额投入碰运气
- **经济周期**：繁荣/正常/衰退，影响整体市场

### 6. 📈 技能成长系统（12种技能）
- 编程、烹饪、健身、社交、音乐、绘画、写作、商业、驾驶、钓鱼、种植、谈判
- 技能通过实践获得经验，用进废退
- 技能影响可用行动和效果
- 技能组合解锁新可能（如编程+商业=创业）

### 7. 📰 事件引擎
- 不是预设脚本，而是条件触发系统
- 根据角色状态+环境+概率动态生成事件
- 事件类型：经济类、社交类、身体类、灵感类、天气类、随机奇遇
- 连锁反应：一个事件可能引发后续事件

## 三、特色玩法

### 🌱 生活空间系统
- 住所类型（合租/独租/公寓/别墅）
- 阳台/花园种植（菜/花/水果），受季节和天气影响
- 宠物养殖（猫/狗/鱼），减少孤独感
- 家具装修，影响生活品质
- 厨房烹饪，自己做饭省钱+技能成长

### 🚗 出行系统
- 步行/公交/打车/自驾/骑行
- 车辆购买和养护
- 通勤时间根据距离和交通方式计算

### 📱 手机系统
- 社交APP：看NPC朋友圈、收消息
- 外卖APP：不想做饭就点外卖
- 招聘APP：找工作
- 新闻APP：获取游戏世界资讯（影响投资决策）
- 购物APP：网购

### 🎣 休闲系统
- 钓鱼：不同水域不同鱼种，装备系统
- 旅行：去其他城市，增长见识
- 娱乐：看电影、打游戏、逛街

### 💼 职业系统
- 求职流程：浏览→投简历→面试→入职
- 多种职业选择，能力/学历/人脉影响结果
- 工作态度选择（认真/摸鱼/社交），影响绩效
- 副业/兼职：外卖、摆摊、直播、写作投稿、家教
- 创业：开店经营模拟

### 🎰 随机乐趣
- 彩票系统（每周开奖）
- 跳蚤市场/二手交易
- 随机奇遇

## 四、UI设计风格

- **不要像素风**，采用现代扁平插画风 + 卡片式交互
- 整体色调温暖，类似 ins/小红书 生活感
- 地图用精致的扁平插画风格
- 交互用卡片点击，手机友好
- 状态面板用仪表盘风格，数据可视化
- 场景用大幅插画背景 + 动效

## 五、技术方案

| 模块 | 技术 |
|------|------|

| UI | HTML5 + CSS3 动画 |
| 引擎 | JavaScript 模块化 |
| 存储 | localStorage 本地多存档 |
